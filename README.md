
## 绿点支付 PHP 接入示例程序

### 说明

绿点支付为独立开发者提供免签即时到账收款接口，只需个人支付宝或微信账号即可收款，极低费率，快捷接入，安全稳定不漏单。详情请访问：[https://greenyep.com](https://greenyep.com)

### 运行

> 请修改 config.php 文件，替换商户 UID 和 Token。

- 将程序部署至公网服务器。
- 访问 index.php 文件，即可进行下单测试。
